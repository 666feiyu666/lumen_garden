# 《萤光花园》十页说明书

此稿按十个页面单元编写，可继续排版为 PDF、小册子或展示板。正式排版时可加入游戏截图、花芽规律示意图和月夜花园概念美术。

## 第 1 页：封面

**萤光花园 / Lumen Garden**  
**副标题：A Puzzle of Blooming Light / 微光盛放之谜**

在静静的夜色中，一枚金色灯灵来到沉睡的花圃。它走过的位置会改变花芽彼此传递的微光。请在黎明之前，让花园盛开成恰好的模样。

画面建议：靛蓝夜园、圆石花径、绿色小花芽、温暖金色灯灵和远处晨光。

---

## 第 2 页：这是一款怎样的游戏

《萤光花园》是一款短局回合制解谜游戏：

- 每则谜题拥有固定花圃与初始花芽。
- 灯灵每走一步或停留一次，花芽都会同时生长一轮。
- 可用步数用尽时，盛开数量达到目标即可解开谜题。
- 玩家寻找的不只是路线，也是一次恰好的光与生长的安排。

---

## 第 3 页：主菜单与模式选择

启动游戏后，你会先看到简洁主菜单：`花园引导`、`种植花谱` 与 `设置`。选择花园引导后可以进入无压力的 `GUIDE 月夜入门` 与第一册五则自由演化谜题；选择种植花谱后可以游玩从稳定花床到自行迁移飞船的五张经典花谱。

| 信息 | 含义 |
| --- | --- |
| 花圃 | 谜题棋盘尺寸 |
| 可用步数 | 黎明前可行动的次数 |
| 盛开目标 | 最终应留下的花芽数量 |

| 按键 | 菜单功能 |
| --- | --- |
| 鼠标左键 | 点击模式、关卡或设置项 |
| `Up/Down` 或 `W/S` | 选择模式或关卡 |
| `Enter` 或 `Space` | 确认选中内容 |
| `G` / `0` / `T` | 进入花园引导选择页 |
| `P` | 进入种植花谱选择页 |
| `Esc` / `Q` | 在主菜单退出；`Esc` 也用于返回上一层 |

## 第 4 页：谜题画面

| 区域 | 你会看到什么 |
| --- | --- |
| 左侧花圃 | 土格棋盘、绿色花芽、金色灯灵与生长变化 |
| 右侧花园记录 | 盛开花芽、盛开目标、黎明倒计时、生长轮次、提示 |

阅读重点：

- 金色灯灵是你的当前位置。
- 绿色花芽是最终要计数的对象。
- 达到目标后仍需坚持到黎明结算，后续生长可能改变数量。

## 第 5 页：操作

| 按键 | 谜题中的作用 |
| --- | --- |
| `W/A/S/D` 或方向键 | 移动灯灵一格，并生长一轮 |
| 点击灯灵/相邻土格 | 停留或移动灯灵，并生长一轮 |
| `Space` | 灯灵停留，并生长一轮 |
| `R` | 重试当前谜题 |
| `Esc` | 返回当前模式的选择页 |
| `Enter` | 结算后重试当前谜题 |

灯灵不能走出花圃，也不能走到已有花芽的土格；这样的动作不会消耗步数。

## 第 6 页：花芽的规律

每一枚花芽会观察周围八格中的微光伙伴：

| 情形 | 下一轮变化 | 直观理解 |
| --- | --- | --- |
| 盛开花芽有 0 或 1 位伙伴 | 熄灭 | 太孤单 |
| 盛开花芽有 2 或 3 位伙伴 | 保持盛开 | 恰好相伴 |
| 盛开花芽有 4 位或更多伙伴 | 熄灭 | 光太拥挤 |
| 空土格恰好得到 3 束微光 | 萌出新芽 | 微光汇聚 |

这套规律源自康威生命游戏，并在花园世界中转化为可以观察的生长节奏。

---

## 第 7 页：灯灵为何重要

金色灯灵具有三条特殊性质：

- 它始终发光，不会随生长轮次熄灭。
- 它会为周围土格和花芽提供一束微光。
- 它不是花芽，因此不计入盛开目标。

你不是在直接栽种花朵，而是在为它们提供刚刚好的光。靠近一处空地可能催生新芽，靠近已经拥挤的花丛也可能让其中一些光芒熄灭。

## 第 8 页：月夜入门

第一次游玩建议从 `GUIDE 月夜入门` 开始：

| 引导 | 你会亲手看到什么 |
| --- | --- |
| 孤芽熄灭 | 没有伙伴的花芽暗下去 |
| 相伴常明 | 小花丛保持原样 |
| 新芽盛放 | 三位伙伴令空地萌芽 |
| 灯灵引路 | 灯灵成为决定新芽的一束光 |
| 小谜题 | 用两轮行动达成盛开目标 |

入门完成后按 `Enter` 即可进入 `AWAKEN 初光`。

## 第 10 页：美术与未来制作

当前方向是一座温暖的月夜花园：

- 背景使用靛蓝夜色、草地、露水和稀疏萤火。
- 灯灵使用醒目的暖金色，与绿色花芽自然区分。
- 花芽盛放与熄灭应清楚可见，同时保持柔和感。
- UI 像一本轻盈的花园手记，而不是实验仪器。

`星图守望 / Starwatch` 作为候选视觉主题保留，但当前版本优先完成萤光花园的试玩、美术和谜题扩展。

当前还提供统一 `10 x 10` 的种植花谱第一册：`P01 BLOCK`、`P02 BLINKER`、`P03 BEACON`、`P04 GLIDER` 与 `P05 LWSS`。灯灵在静止花圃里以方向键/`WASD` 或点击相邻土格移动，按 `P` 或点击“播种”按钮种下有限花种；花圃内点击灯灵不会落种。花种全部种下后，点击“开始”或按 `Space` / `Enter` 放手生长。验证期间可以暂停或单步查看；成功成谱后，花圃循环演示已验证的安全片段，可暂停或从开头重播。最后两题会分别斜向和横向迁移，要求你阅读变化后的位置。
