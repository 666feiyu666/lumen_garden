# `00 GUIDE / 月夜入门` 设计

## 目的

《萤光花园》以康威生命游戏为规则骨架，但玩家看到的是花芽彼此传递微光的月夜谜题。`GUIDE` 不承担难度挑战，而是让玩家亲手理解：每一次灯灵移动或停留，花园都会生长一轮。

## 教学目标

完成入门后，玩家应当理解：

1. 每次移动或停留都会触发生长。
2. 花芽周围微光太少或太多会熄灭。
3. 拥有 2 或 3 位伙伴的花芽会继续盛开。
4. 空土格得到恰好 3 束邻近微光会萌出新芽。
5. 金色灯灵不计入盛开数量，但它能提供邻近微光。

## 已实现流程

| 段落 | 标题 | 花圃 | 操作 | 要点 |
| ---: | --- | ---: | --- | --- |
| 0 | 遇见灯灵 | 5 x 5 | `Enter` | 金色灯灵与绿色花芽的身份 |
| 1 | 孤芽熄灭 | 5 x 5 | `Space` | 少于 2 位伙伴会熄灭 |
| 2 | 相伴常明 | 5 x 5 | `Space` | 2-3 位伙伴会维持盛开 |
| 3 | 新芽盛放 | 5 x 5 | `Space` | 三束微光催生新芽 |
| 4 | 灯灵引路 | 6 x 6 | `Left/A` | 灯灵可补上关键微光 |
| 5 | 第一则小谜题 | 6 x 6 | 自由行动 | 两轮内令 3 枚花芽盛开 |
| 6 | 完成 | - | `Enter` | 前往 `01 AWAKEN 初光` |

## 屏幕反馈

| 元素 | 表现 |
| --- | --- |
| 灯灵 | 暖金色发光标记，与花芽明显区分 |
| 花芽 | 绿色花形标记；新芽变亮，熄灭时淡出 |
| 观察土格 | 明亮边框 |
| 周围伙伴格 | 淡色边框，提示八邻域 |
| 灯灵移动目标 | 暖金边框与 `MOVE` 标签 |
| 指引面板 | 说明当前规律、伙伴数量与下一步按键 |

## 小谜题

最后一段第一次引入正式谜题语言：

- 两轮生长机会。
- 盛开目标为 `3`。
- 一条已验证解法是 `LEFT` 后 `WAIT`。
- 失败后可用 `R` 立即重试，不产生惩罚。

完成后进入 `AWAKEN 初光`，玩家已经从理解规律转为主动解谜。

## 后续试玩问题

| 观察项 | 判断标准 |
| --- | --- |
| 玩家是否区分灯灵与花芽 | 能说出灯灵不计入目标、但会提供微光 |
| 玩家是否理解熄灭 | 能解释伙伴太少/太多会熄灭 |
| 玩家是否理解萌芽 | 能指出三位邻近伙伴会带来新芽 |
| 文案是否自然 | 不需要使用“细胞”“实验”仍能说明规则 |
