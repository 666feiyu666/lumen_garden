# Phase 1 原型文档归档

本目录保存《萤光花园》第一阶段的可运行原型设计文档。Phase 1 的重点是验证灯灵、康威规则、花园谜题、种植花谱和基础美术音频接入。

这些文档仍然有效，但它们描述的是“原型基线”，不是 Phase 2 第一章的最终形态。

| 文档 | 用途 |
| --- | --- |
| [单页说明书](one-page-manual.md) | 给首次接触项目的人快速理解并开始游戏 |
| [十页说明书](ten-page-manual.md) | 可继续排版为展示或交付手册的长版说明 |
| [游戏流程表](game-flow.md) | 描述原型界面流转、单局循环与教学流程 |
| [游戏设计文档](game-design-document.md) | 统一原型玩法、美术、技术、测试与版本目标 |
| [月夜入门设计](tutorial-level-design.md) | 把康威规则转化为可玩的引导体验 |
| [戏剧方向决策](narrative-directions.md) | 记录萤光花园主方案与星图守望备选方案 |
| [种植花谱模式设计](classic-mode-design.md) | 规定灯灵播种、经典图案验证与自由演化玩法的边界 |

## Phase 1 已验证内容

- 灯灵作为玩家身份和额外微光的规则表达。
- 统一 `10 x 10` 棋盘下的花园谜题可读性。
- `Block`、`Blinker`、`Beacon`、`Glider`、`LWSS` 的经典行为验证。
- 显式播种、撤回、自动验证和成谱演示。
- 绘本风美术方向、灯灵双形态和基础音乐接入。

## Phase 2 继承原则

- 继承康威 `B3/S23` 作为世界底层物理。
- 继承灯灵、花芽、经典图案和温暖孤独的叙事基调。
- 不再把“单一花圃谜题册”作为最终结构，转向多房间遗迹迷宫和第一章冒险流程。
