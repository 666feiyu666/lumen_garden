# Phase 2 设计总览

Phase 2 的目标是完成《萤光花园》第一章的最小完整版本：灯灵从荧光森林的遗迹棋盘中醒来，学习普通植物与机关植物的差异，连续通过三个房间，最终离开森林迷宫并回到灯漫村。

当前实现已经落地第一版可运行闭环：静态教程、开场漫画、三房间线性森林、结尾漫画、设置弹窗和荧光花园占位入口。接下来应继续打磨房间谜题质量、机关反馈、章节回报和荧光花园后续玩法。

## 阶段目标

| 目标 | 说明 |
| --- | --- |
| 第一章最小闭环 | 从普通植物房、机关植物房到混合房，完成一段短而完整的森林迷宫 |
| 房间式迷宫 | 每个房间是一张棋盘；第一章先采用 `房间一 -> 房间二 -> 房间三` 的线性结构 |
| 两类植物 | 普通植物和机关植物都遵循康威生命游戏规则，但启动时机和关卡作用不同 |
| 图案功能化 | `Glider` 先作为机关植物代表，用残缺形状补种后触发远处机关 |
| 章节回报 | 第三房间出口触发结尾漫画，随后进入荧光花园解锁/占位页 |

## 当前实现范围

- 首页入口：`教程`、`荧光森林`、`荧光花园`、`设置`。
- 教程：静态生命游戏规则页，使用 `tutorial_menu.png`。
- 荧光森林：点击后先播放 `Introduction_1.png` 开场漫画，再进入房间一。
- 房间一：普通植物房，普通植物随灯灵移动或等待演化，灯灵作为邻居参与判断。
- 房间二：机关植物房，残缺 `Glider` 在补种并启动前休眠，启动后随行动演化并触发机关。
- 房间三：普通植物和机关植物混合，出口触发 `end_1.png` 结尾漫画。
- 荧光花园：当前显示占位页，使用 `garden.png`，后续扩展为大本营与花谱空间。
- 设置：`Esc` 打开设置；关卡内设置提供退出关卡，并在本次运行内记录森林房间进度。

## 设计边界

- 借鉴《咩咩启示录》的“房间串联、短程探索、节点节奏”，但第一章第一版只做三房间线性流程。
- 第一章以手工设计为主，不做随机房间、复杂分支或完整经营系统。
- 每个房间都应是可读的棋盘谜题，而不是装饰性走廊。
- 玩家失败或悔棋应服务于思考，不制造惩罚性重复。
- Phase 1 的花园谜题与种植花谱保留为底层验证资产，后续可逐步转译为荧光花园、花谱或章节奖励内容。
