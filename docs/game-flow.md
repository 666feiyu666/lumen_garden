# 《萤光花园》游戏流程表

## 总体流程

| 阶段 | 状态 | 玩家操作 | 画面反馈 | 下一状态 |
| --- | --- | --- | --- | --- |
| 启动 | 现有 | 运行 `main.py` | 简洁主菜单：花园引导、种植花谱、设置 | 主菜单 |
| 设置 | 现有 | 点击设置，选择窗口尺寸/音量/静音 | 弹窗覆盖主菜单 | 主菜单 |
| 花园引导选择 | 现有 | 点击模式入口，再选入门或谜题 | 展示统一 `10 x 10` 第一册的步数、盛开目标和短文 | 月夜入门/花园谜题 |
| 月夜入门 | 现有 | 按提示停留或移动灯灵 | 高亮伙伴格，解释熄灭与盛放 | 入门完成/重试 |
| 谜题选择 | 现有 | 上下选择，或按 `1-5` 直达 | 展示花圃、步数、盛开目标和短文 | 花园谜题 |
| 花园谜题 | 现有 | 移动或停留 | 生长一轮，更新花园记录 | 谜题/结算 |
| 无效行动 | 现有 | 越界或走向已有花芽 | 提示原因，不消耗步数 | 花园谜题 |
| 谜题解开 | 现有 | 点击或使用 `Enter/R`、`Esc` | 展示花园盛放 | 重试/花园引导选择 |
| 未能成景 | 现有 | 点击或使用 `Enter/R`、`Esc` | 展示月夜散场 | 重试/花园引导选择 |
| 谜题册进度 | 计划 | 解开谜题 | 记录路径与解锁新谜面 | 花园引导选择 |
| 种植花谱选择 | `P01-P05` 已实现 | 选择静物、周期或飞船花谱 | 显示花种预算、目标行为与验证代数 | 播种阶段 |
| 播种阶段 | `P01-P05` 已实现 | 键盘/相邻格点击移动，以 `P`/按钮播种或 `X`/按钮撤回；花种用完后以 `Space`/`Enter`/按钮确认 | 花圃保持静止，未完成提交会提示剩余花种 | 播种阶段/生长验证 |
| 生长验证 | `P01-P05` 已实现 | 确认开始，暂停/继续、单步查看或重置 | 灯灵退场，经典图案自动播放纯规则生长 | 成谱演示/重新播种 |
| 成谱演示 | `P01-P05` 已实现 | 暂停/继续、从片段开头重播或重置 | 循环播放已验证安全片段，成功判定保持不变 | 成谱演示/播种阶段 |

所有菜单入口、关卡卡片和主要操作按钮均支持鼠标点击。花园谜题中可点击灯灵所在格停留、点击相邻土格移动；种植阶段的花圃点击只负责向相邻土格移动，点击灯灵仅提示键盘/按钮动作，播种与开始仍使用明确动作。

## 单步循环

| 序号 | 发生的事 | 数据变化 | 表现目标 |
| ---: | --- | --- | --- |
| 1 | 等待行动 | 无 | 灯灵、花芽、盛开目标与黎明倒计时清楚可见 |
| 2 | 验证移动 | 检查边界和花芽占用 | 无效行动给出柔和拒绝提示 |
| 3 | 灯灵移动/停留 | 更新金色灯灵位置 | 暖金光点平滑移动或呼吸 |
| 4 | 花园生长 | 依据伙伴数量生成下一轮花芽 | 新芽亮起，熄灭花芽散落 |
| 5 | 消耗步数 | 剩余步数减少、生长轮次增加 | HUD 有轻量反馈 |
| 6 | 黎明检查 | 最后一步比较盛开目标 | 花园盛放或月夜散场 |

## 种植花谱循环（`P01-P05` 已实现）

| 序号 | 发生的事 | 数据变化 | 表现目标 |
| ---: | --- | --- | --- |
| 1 | 阅读花谱目标 | 无 | 明确花种数量、经典行为与验证代数 |
| 2 | 灯灵移动 | 更新位置 | 移动不触发生长，突出这是布置阶段 |
| 3 | 以 `P`/按钮播种或撤回 | 更新玩家播下的花芽与花种余额 | 预置花芽和新种花芽可辨认，误按 `Space` 不种花 |
| 4 | 完成播种并以 `Space`/`Enter`/按钮确认生长 | 花种未用完则拒绝；否则锁定初始布局 | 明确提醒剩余花种，完成后灯灵退场 |
| 5 | 纯规则演化 | 不含灯灵邻居地自动推进代数 | 支持暂停与单步，显示稳定、周期或迁移进度 |
| 6 | 行为验证 | 检查标准图案目标 | 成功进入成谱演示，失败返回重新播种 |
| 7 | 成谱演示 | 循环显示验证过程中记录的安全帧，不改变模型胜负 | 动态花谱持续可见，飞船不会为展示而撞边 |

## 首次体验

| 顺序 | 内容 | 玩家理解 |
| ---: | --- | --- |
| 0 | 默认选中 `GUIDE 月夜入门` | 此处可先学习，无压力 |
| 1 | 孤芽熄灭 | 太孤单的花芽无法维持微光 |
| 2 | 相伴常明 | 两三位伙伴能让花芽延续 |
| 3 | 新芽盛放 | 三束微光会让空地萌芽 |
| 4 | 灯灵引路 | 玩家位置本身是关键影响 |
| 5 | 第一则小谜题 | 将规则用于目标解谜 |
| 6 | `AWAKEN 初光` | 进入第一则正式谜题 |

## 当前谜题册

| 顺序 | 谜题 | 主题 |
| ---: | --- | --- |
| 00 | GUIDE 月夜入门 | 学习微光规律 |
| 01 | AWAKEN 初光 | 在 `10 x 10` 花圃中第一次主动介入 |
| 02 | SPROUT 新芽 | 连续照料并延续新芽 |
| 03 | CHORUS 合鸣 | 配合停留协调相遇花群 |
| 04 | MOONBED 月圃 | 以主动停驻选择照料时机 |
| 05 | DAWN 黎明 | 在同一尺度中完成路线收束 |

正式第一册的五则花园谜题均采用 `10 x 10`；可变棋盘尺寸继续保留给未来的特殊内容，不作为常规难度阶梯。
