# 《萤光花园》文档中心

本目录用于把可运行原型整理成可沟通、可测试、可继续制作的游戏方案。

## 当前基线

- 当前已实现：`00 GUIDE` 月夜入门、统一 `10 x 10` 的五则正式花园谜题、包含 `Block` / `Blinker` / `Beacon` / `Glider` / `LWSS` 的五则种植花谱、`P` 显式播种与未完成播种拦截、自动验证和成功后的成谱循环演示、菜单/HUD/结果界面、基础音频接口。
- 当前未实现：正式图片素材、正式音乐音效、独立花谱收藏/自由回看页、谜题册进度与存档、入门完成记录。
- 当前创作方向：采用 **萤光花园 / Lumen Garden**；**星图守望 / Starwatch** 作为保留候选方案。
- 下一版建议重点：试玩固定尺度的双模式各五关，校准从稳定、周期到位移的花谱节奏，并验证显式播种及 `LWSS` 成谱演示的阅读负担。

## 制作顺序建议

1. 试玩统一 `10 x 10` 的花园第一册，检验五关是否形成可感知的决策梯度。
2. 试玩 `P01-P05` 的 `P` 键播种、未完成提示与成谱循环演示反馈。
3. 调整 `Beacon`、`Glider` 与 `LWSS` 形成的动态阅读难度坡度。
4. 在双模式十关成形后，迭代正式美术与音乐。
5. 后续扩展花谱回看、进度存档、特殊道具与附加关卡。
