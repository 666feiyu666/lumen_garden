# 《萤光花园》单页说明书

## 游戏概念

你带领一枚温暖的金色灯灵，在月夜花圃中解开微光谜题。每移动一步或停留一轮，绿色花芽都会依照身边伙伴的数量成长、盛放或熄灭。请在黎明到来前，让恰好数量的花芽留在园中。

## 如何解谜

- **金色灯灵**：由玩家控制，不会熄灭，也不计入盛开目标。
- **绿色花芽**：会盛放或熄灭，计入当前花芽数量。
- **每一步**：移动灯灵一格，或原地停留，然后花园立即生长一轮。
- **黎明结算**：可用步数耗尽时，盛开花芽数量处于目标范围内即解开谜题。

## 花芽生长规律

| 当前状态 | 周围微光伙伴数量 | 下一轮结果 |
| --- | ---: | --- |
| 盛开花芽 | 2 或 3 | 继续盛开 |
| 盛开花芽 | 少于 2 或多于 3 | 渐渐熄灭 |
| 空土格 | 恰好 3 | 萌出新芽 |

灯灵也能提供一束邻近微光，因此它所停留的位置能够改变花圃的下一轮模样。

## 种植花谱

种植花谱第一册包含五张 `10 x 10` 花圃：`BLOCK / 四叶眠床` 教授稳定花床，`BLINKER / 摇曳花枝` 与 `BEACON / 双灯花台` 展示周期变化，`GLIDER / 流光种` 与 `LWSS / 晨风舟` 展示自行迁移。你用有限花种补全初态；花种全部种下后，灯灵退到一旁，花谱自动播放纯粹的康威规则并接受行为验证。验证成功后可循环观看已验证的安全片段。

## 操作

| 按键 | 功能 |
| --- | --- |
| 鼠标左键 | 选择模式/关卡、点击按钮，或在花圃中操作灯灵 |
| `G` / `0` / `T` | 从主菜单进入花园引导 |
| `1` 至 `5` | 在花园引导中直接进入五则谜题 |
| `P` | 从主菜单进入种植花谱 |
| `W/A/S/D` 或方向键 | 在花园谜题中移动灯灵，并生长一轮 |
| `Space` | 停留，并生长一轮 |
| `R` | 重试当前谜题或当前指引 |
| `Esc` | 对局返回模式选择；再次返回主菜单；在主菜单退出 |

在种植花谱中，方向键或点击相邻格移动不触发生长；花圃内点击灯灵也不会落种或开演。`P` 或“播种”按钮种下花种，`X` / `Backspace` 或按钮撤回花种。花种未种完时，`Space` / `Enter` 会提醒继续播种；全部种下后以 `Space` / `Enter` 或“开始”按钮启动验证。验证过程中 `Space` 可暂停/继续，`Enter` 可单步查看；成谱演示中 `Space` 可暂停/继续，`Enter` 可重新播放片段。

| 01 AWAKEN 初光 | 10 x 10 | 3 | 3 |

| 03 CHORUS 合鸣 | 10 x 10 | 6 | 19 |
| 04 MOONBED 月圃 | 10 x 10 | 7 | 19 |
| 05 DAWN 黎明 | 10 x 10 | 9 | 25 |

第一册统一使用 `10 x 10` 花圃；难度来自初始花群、灯灵路线、停留时机与收束目标，而不是棋盘逐关扩大。

## 推荐起点

首次游玩请选择 `00 GUIDE 月夜入门`。它会通过孤芽熄灭、花芽相伴、新芽盛放和灯灵引路四段演示，带你理解每一轮花园为何改变。
