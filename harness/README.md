# Harness Prompts

本目录存放可直接交给编码 agent 的阶段性执行提示词，与 `docs/` 中的游戏设计、流程和说明文档分开管理。

提示词依照执行顺序以两位编号排列。新阶段应新增下一编号文件，而不覆盖已执行或待检查的阶段范围。

| 顺序 | 提示词 | 用途 | 状态 |
| ---: | --- | --- | --- |
| 01 | [种植花谱原型开发](01-planting-bloom-prototype.md) | 实现灯灵播种、纯规则验证的种植花谱最小可玩原型 | 已完成 |
| 02 | [主菜单与鼠标交互](02-main-menu-and-mouse-ui.md) | 简化入口、增加设置弹窗与鼠标操作 | 已完成 |
| 03 | [固定花圃与花谱教学播放](03-fixed-garden-and-planting-playback.md) | 统一正式花圃、重制花园五关并打磨种植教学验证 | 已完成 |
| 04 | [经典动画花谱扩展](04-classic-pattern-expansion.md) | 移植周期与飞船图案，完成种植花谱第一册五关 | 已完成 |
| 05 | [播种确认与花谱循环展示](05-planting-input-and-pattern-showcase.md) | 明确播种输入、防止提前验证，并让成功花谱可持续观看 | 已完成 |
