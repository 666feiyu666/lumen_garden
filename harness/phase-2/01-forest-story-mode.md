# Phase 2 Harness Prompt 01: 荧光森林三房间剧情模式

## 任务目标

基于 `docs/phase-2/chapter-one-design.md` 与 `docs/phase-2/room-and-mechanism-design.md`，实现 Phase 2 第一章的最小可运行闭环。当前版本不追求最终关卡难度，而是验证首页入口、开场漫画、三房间线性剧情流程、普通植物与机关植物的基本时序。

## 首页更新

首页入口调整为四项：

```text

荧光花园
```

- `教程`：进入静态生命游戏规则页，不再进入互动教程流程。

- `荧光花园`：当前显示“正在开发”，不进入旧种植花谱菜单。

旧的 Phase 1 花园谜题和种植花谱代码暂时保留，作为规则和回归资产，不从首页主入口暴露。

## 静态教程

- 教程只展示生命游戏的四条基础规则和小棋盘示意。
- 教程不要求玩家移动灯灵，也不承担动态教学。
- 灯灵如何影响普通植物、机关植物如何休眠和启动，应在荧光森林房间中通过游玩学习。
- 按 `Enter` / `Space` / 鼠标点击返回首页。

- 点击“荧光森林”后进入漫画场景。
- 漫画加载 `assets/sprites/Introduction_1.png`。

- 按 `Enter` / `Space` 或鼠标点击进入房间一。

- 第三个房间到达出口并通关时，不直接跳回首页。
- 先显示 `assets/sprites/end_1.png` 作为第一章结尾漫画。
- 按 `Enter` / `Space` / 鼠标点击后，进入荧光花园已解锁但正在开发的提示页。

## 第一章房间结构

第一章采用线性三房间：

| 房间 | 主题 | 实现要求 |
| ---: | --- | --- |
| 1 | 普通植物房 | 只有普通植物。灯灵从棋盘下部进入；普通植物挡住通路或出口；玩家移动/等待时普通植物演化，灯灵作为邻居参与普通植物判断。出口在左边缘或上边缘。 |
| 2 | 机关植物房 | 有残缺 `Glider` 和一枚可补回的种子。未启动前机关植物休眠，不随移动演化；补种并启动后随灯灵移动/等待演化，触碰占位机关后开门。 |
| 3 | 混合房 | 同时包含普通植物和机关植物。普通植物始终随行动演化；机关植物未启动前休眠，启动后随行动演化。通关后返回首页并解锁/提示荧光花园。 |

## 规则时序

- 普通植物：每次灯灵有效移动或等待，按 `B3/S23` 演化一代，并将灯灵作为额外邻居。
- 机关植物：未启动前保持休眠；补齐并启动后，按 `B3/S23` 演化。
- 已启动机关植物：每次灯灵有效移动或等待推进一代。
- 第一版允许用简化机关形状代替正式月光接收器。
- 第一版不实装黑水、`LWSS`、大炮、复杂背包或正式图鉴。

## UI 要求

- 关卡界面沿用“棋盘 + 黑板”结构。
- 黑板显示当前房间名、目标、普通植物/机关植物状态、操作提示。
- 棋盘中应能区分灯灵、普通植物、休眠机关植物、已启动机关植物、种子、接收器和出口。
- 房间出口位于左边缘或上边缘。

## 测试要求

- 覆盖点击“荧光森林”进入漫画，再进入房间一。
