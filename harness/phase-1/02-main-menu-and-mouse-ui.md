# 02 主菜单与鼠标交互 Harness Prompt

## 任务目标

将《萤光花园》的入口界面从直接展示全部关卡，调整为符合常见游戏结构的简洁主菜单，并让玩家可以使用鼠标完成主要操作。

## 已确认设计决定

- 初始界面只提供主要入口：`花园引导`、`种植花谱`、`设置`。
- `花园引导` 进入当前教程与五则自由演化谜题的选择页。
- `种植花谱` 进入已实现的 `P01 BLOCK / 四叶眠床` 以及未来种植内容的选择页。
- `设置` 以主菜单上的弹窗形式提供窗口尺寸与音量/静音设置。
- 保留当前键盘操作，同时为菜单和可玩场景补充鼠标点击。

## 玩家可感知验收

- 启动后不再直接面对长关卡列表，而是看到三个简洁入口。
- 点击两个模式入口可以进入对应关卡选择页，并能点击卡片开始内容。
- 点击设置能够打开/关闭弹窗，切换窗口大小并调节音量或静音。
- 自由演化中可以点击相邻格移动灯灵、点击灯灵所在格停留，并点击返回/重试。
- 种植花谱中可以点击相邻格移动灯灵，并点击播种、撤回、开始生长、推进、重置和返回按钮。

## 验证要求

- 既有规则与种植规则测试继续通过。
- 增加 dummy SDL 界面烟雾测试，覆盖鼠标点击进入模式、开始种植谜题及设置弹窗基本操作。

## 执行状态

已于 2026-05-26 完成：主菜单分层、设置弹窗、窗口/声音控制、菜单与核心玩法鼠标交互，以及 dummy SDL 烟雾测试。
