# 03 固定花圃与花谱教学播放 Harness Prompt

## 任务目标

将《萤光花园》的两种常规玩法建立在统一、易读的第一册花圃规格上，并把 `P01 BLOCK / 四叶眠床` 从可运行原型打磨为种植花谱教学关。

本任务承认此前的五则花园谜题只验证了功能闭环和参考路径，没有经过难度体验验证；因此不保留其逐关扩大的棋盘布局，而重新制作正式第一册数据。

## 已确认设计决定

- 正式第一册中的花园谜题与种植花谱常规关卡均采用 `10 x 10` 花圃。
- 底层继续保留可变棋盘尺寸支持，为未来特殊关卡服务；主线难度不依赖棋盘扩大。
- 花园谜题继续使用灯灵参与每一轮邻居计算的规则，并以黎明时盛开数量判定结果。
- 花园谜题的难度变量为初始花芽、灯灵路线、停留时机、行动轮数和目标花量。
- 种植花谱继续坚持“先种植，后验证”：验证阶段灯灵不参与邻居计算，目标依经典行为判定。
- `P01 四叶眠床` 作为种植花谱教学关，说明移动不生长、播种构造初态、放手后由纯规则验证。
- 花谱验证应自动播放，并支持暂停与单步查看，为后续 `Blinker` 和 `Glider` 动态目标奠定交互基础。
- 道具、花谱五关全部内容和正式美术/音乐迭代不在本轮实现范围内。

## 交付范围

### M1：固定尺度的花园第一册

- 将 `AWAKEN` 至 `DAWN` 五则花园谜题全部重制为 `10 x 10`。
- 五关依次承接：第一次介入、连续照料、花群交汇、停留取舍、长路线收束。
- 为每关提供至少一条规则层可复测的参考解法，并确认只停留无法解开五关。
- 文档明确：参考路径用于功能回归，难度曲线仍需实际试玩校准。

### M2：种植教学关正式化

- 将 `P01 BLOCK / 四叶眠床` 迁移至 `10 x 10`。
- 保持一枚花种补齐稳定四叶花床的清楚教学任务。
- 在界面中说明播种阶段与验证阶段的规则区别。

### M3：花谱验证播放反馈

- 点击“开始”或按 `Enter` 后自动播放纯规则验证。
- 验证阶段支持播放/暂停和单步推进。
- 结算文案依据当前图案行为生成，不硬编码为某一关的稳定结果。
- 不在本轮添加可玩 `Blinker` 或 `Glider` 关卡；现有规则测试继续保护它们的标准行为。

### M4：文档、测试与路线图

- 更新说明书、流程与设计文档中的棋盘规格、交互和当前状态。
- 增加固定棋盘规范与自动播放 UI 烟雾测试。
- 记录后续优先级：
  1. 试玩并调优双模式当前内容，尤其是新花园五关的难度梯度。
  2. 完成种植花谱 `P02-P05`，以 `Blinker` 与 `Glider` 作为核心动态目标。
  3. 对双模式十关进行一次美术与音乐迭代。
  4. 后续再扩展回看、进度保存、道具和特殊关卡。

## 验收标准

- 五则正式花园谜题数据均为 `10 x 10`，且参考路径可完成目标。
- 五关等待到底均不能误判为成功。
- `P01` 为 `10 x 10`，能够完成播种、启动纯规则验证并成功结算。
- 启动花谱验证后可自动播放，玩家可以暂停并单步查看。
- 原有教程、经典图案纯规则检测与菜单/鼠标操作仍通过测试。
- 文档不得把尚未试玩验证的难度曲线、未实现的花谱关卡或美术音乐描述为完成。
